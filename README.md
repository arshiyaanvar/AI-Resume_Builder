# AI Resume Builder (GenAI Project)

## 📌 Description
A simple Generative AI application that generates professional resume bullet points based on a given job role using Groq LLM API.

## 🚀 Features
- Generates resume bullet points instantly
- Uses LLM (LLaMA 3) via Groq API
- CLI-based application

## 🛠️ Tech Stack
- Python
- Groq API
- VS Code

## ▶️ How to Run

1. Create virtual environment:
   python -m venv venv
   venv\Scripts\activate   (Windows)

2. Install dependencies:
   pip install -r requirements.txt

3. Add API key in .env file

4. Run:
   python app.py

## 💡 Example
Input: Data Analyst  
Output: AI-generated resume bullet points
